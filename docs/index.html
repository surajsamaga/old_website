Projects

Graph Neural Networks

Research Internship, SARC ILP

Tasks such as modeling physics systems, learning molecular fingerprints etc. require dealing with graph data which contains rich relation information among elements. Graph neural networks (GNNs) are connectionist models that capture the dependence of graphs via message passing between the nodes of graphs. I attempt to tackle the problem of decreasing performance of these GNNs with increasing architectural depth (and hence complexity).
